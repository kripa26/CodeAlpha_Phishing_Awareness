# Phishing Awareness Training Module

**CodeAlpha Cybersecurity Internship — Task 2**  
**Author:** Kripa SB

---

## What is This?

An interactive, web-based training module designed to educate users about phishing attacks — how they work, how to spot them, and how to stay safe online. Built as a single-page application with a premium dark cybersecurity-themed UI.

---

## Features

| Feature | Description |
|---------|-------------|
| **Interactive Email Simulator** | A realistic phishing email with clickable red flags — click each suspicious element to learn why it's dangerous |
| **Phishing Types** | Covers all major types: Email, Spear, Smishing, Vishing, Whaling, and Clone Phishing |
| **Social Engineering Tactics** | Breakdown of psychological tricks attackers use: Urgency, Authority, Fear, Trust, Greed, Helpfulness |
| **Real-World Case Studies** | Google/Facebook $100M scam, Twitter Bitcoin hack, Colonial Pipeline, MGM Resorts |
| **10-Question Quiz** | Interactive quiz with 3 difficulty levels (Easy, Medium, Hard), instant feedback, and score tracking |
| **Answer Review** | Review all your answers with detailed explanations after completing the quiz |
| **Best Practices** | 8 actionable security tips with clear guidance |
| **Responsive Design** | Works on desktop, tablet, and mobile devices |

## How to Run

Simply open `index.html` in any modern web browser:

```bash
# Clone the repository
git clone https://github.com/kripa26/CodeAlpha_Phishing_Awareness.git

# Navigate to the folder
cd CodeAlpha_Phishing_Awareness

# Open in browser (Windows)
start index.html

# Open in browser (Mac)
open index.html

# Open in browser (Linux)
xdg-open index.html
```

No dependencies, no build tools, no server required — just open and learn!
